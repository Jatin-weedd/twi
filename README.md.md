# 🏔️ India TWI Data Gateway (Render)

Secure tokenless resolution gatekeeper for streaming private Topographic Wetness Index (TWI) COG tiles dynamically.

## 🔐 Environment Setup on Render
Set these exact environment keys in your Render service deployment settings:
* `HF_TOKEN`: Read-access token to your private Hugging Face dataset.
* `CLIENT_API_KEY`: Custom string used to authorize your client connections.
